# Connection with arduino

Motor Nema17 38.3x38.3x20__ 

## Motor A4988 controller pins
ENABLE    => D13__
DIRECTION => D4__
STEP      => D5__
GND       => EXT ALIM__
VMOT      => EXT ALIM__
SLEEP     => RESET__
1B        => BLACK__
1A        => GREEN__
2A        => BLUE__
2B        => RED__

## Fingerprint sensor R503
white  => 3.3V__
red    => 3.3V__
yellow => D2__
brown  => D3__
black  => GND__
