# Connection with arduino

Motor Nema17 38.3x38.3x20   

## Motor A4988 controller pins
ENABLE    => D13  
DIRECTION => D4  
STEP      => D5  
GND       => EXT ALIM  
VMOT      => EXT ALIM  
SLEEP     => RESET  
1B        => BLACK  
1A        => GREEN  
2A        => BLUE  
2B        => RED  

## Fingerprint sensor R503
white  => 3.3V  
red    => 3.3V  
yellow => D2  
brown  => D3  
black  => GND
